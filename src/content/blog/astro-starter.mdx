# Welcome to GMC Tourism

This starter combines **Astro**, **Tailwind CSS 4**, and **MDX** to deliver a blazing-fast tourism
experience that looks great on every screen. Out of the box you get optimized metadata, responsive
layouts, and a scalable content architecture powered by `src/content`.

<Callout type="tip" title="Why Astro?">
  Astro ships zero JavaScript by default, letting you craft immersive travel
  experiences without sacrificing performance. When you need interactivity,
  hydrate only the components that matter.
</Callout>
